# 樂譜頁 Layout 計劃（按設計圖）

## 1. 頂部：兩條操作列（深色背景）

唔係分兩區，而係**兩條獨立嘅操作列**，滾動行為唔同：

### 第一條：操作列（固定，唔跟住滾動）

- 深色背景 `#000000` / `#121212`
- **頁面滾動時留喺原位**（sticky 或 fixed）
- 一列圖標：資訊 ⓘ、文件、剪貼簿、心心、加入歌單 (+)、更多（三點）

### YouTube / 歌曲資訊（完全隱藏，頂部 ⓘ 掣先出現並展開）

- 預設**完全唔顯示**；只有撳頂部 資訊 ⓘ 掣先會出現並展開（在 Key 選擇器上方）
- 內有 YouTube 嵌入與歌曲資訊（年份、專輯、作曲、填詞等）；可撳區內收起掣或再撳 ⓘ 收起

### 第二條：Key 選擇器（跟住頁面滾動）

- 深色背景（同上）
- **會沿住頁面一齊滾動**（唔 fixed）
- **上方**：文字「原調: D」「PLAY: C」「Capo 2」（Capo 2 用黃底 pill）
- **下方**（同一條內）：一排圓形 Key 掣（選中 = 黃底黑字，其餘 = 深底淺字）

---

## 2. 歌曲資訊區（Header）

- 左：封面縮圖（方形圓角）
- 右：歌名（白）、歌手（灰）、出譜 pill（黃底黑字「出譜: Kermit Tam」）、星星評分

## 3. 譜面內容區

- 黑底、Section 標題、和弦（黃）、歌詞（白/灰）、等寬字體

## 4. 浮動控制列（可收埋/展開，唔要黃 bar）

- **收埋時**：頁面右下角一個浮動圓形 icon（黃底 #FFD700），撳開展開
- **展開時**：浮動視窗（深色 #121212 卡片），內有「A 字體」橫向滑桿、**自動滾動橫向滑桿**（0=關，1–4=速度）、**主題切換（日間/夜間模式）**、眼睛（隱藏/顯示簡譜）、括號 ( )；可撳 × 收埋
